Instructions

Welcome to the K-Means clustering demo! Here's how to use the application:

  • Step 1: Click on "Generate new dataset" to create random data points.

  • Step 2: Input the number of clusters (2 as minimum) in the input field labeled "Input number of clusters".

  • Step 3: Click on "Apply algorithm" to run the K-Means algorithm and see the clustering result.

  • Step 4: View the information in the table .

The K-Means algorithm will group the points into clusters, and the centroids (representative points) will be marked in black 'X'.
