Instructions

Welcome to the K-Means clustering demo! Here's how to use the application:

  • Step 1: Click on "Generate new dataset" to create random data points.

  • Step 2: Input the number of clusters (between 2 and 8) in the input field labeled "Input number of clusters".

  • Step 3: Click on "Apply" to run the K-Means algorithm and see the clustering result.

  • Step 4: View the information in the table below the canvas (e.g., number of points, iterations, clusters).

The K-Means algorithm will group the points into clusters, and the centroids (representative points) will be marked in black.
